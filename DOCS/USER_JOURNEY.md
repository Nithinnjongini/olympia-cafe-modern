# User Journey - Olympia Cafe & Gyros

This document outlines the intended user journey for customers interacting with the modernized Olympia Cafe web application.

## 1. Discovery & Branding (Landing Page)
- **Visual Impact:** Users arrive at the site and are greeted by high-impact visuals of authentic Greek cuisine and a modern, clean aesthetic.
- **Trust Elements:** The "Est. 1985" badge and Google Review counter immediately establish credibility.
- **Immediate Action:** The "Order Online" button is prominently displayed for repeat customers who want to jump straight to the point.

## 2. Exploration (The Menu)
- **Categorization:** Users navigate to the menu and can easily filter by category (Appetizers, Gyros, etc.) using the sticky navigation.
- **Item Details:** Each menu item features a concise description and clear pricing. popular items are highlighted with a "Popular" badge.
- **Seamless Transition:** Clicking "Add to Cart" or "Order Now" on any item directs the user specifically to the ToastTab ordering platform where the actual transaction occurs.

## 3. Information & Trust (Info Page)
- **Accessibility:** Users looking for the physical location find an interactive map that supports "Get Directions" functionality.
- **Clarity:** Business hours are presented in a clean, high-contrast table, reducing friction for users checking if the restaurant is currently open.
- **Social Connection:** Direct links to Instagram and Facebook allow users to engage with current specials and community updates.

## 4. Catering & Events (Catering Page)
- **Specialized Offering:** Users planning events find clear, per-person pricing for the famous "Gyro Bar."
- **Direct Communication:** The catering page emphasizes direct phone contact, recognizing that event planning often requires personal coordination.

## 5. Mobile Experience
- **Responsiveness:** All interactions are designed with a mobile-first mindset, ensuring the menu is readable and buttons are "finger-friendly" on smartphones.
- **Speed:** Next.js optimizations ensure that navigation between pages feels instantaneous, matching the fast-paced nature of mobile users.
